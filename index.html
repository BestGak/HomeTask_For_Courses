  • git config --global user.name blabla - команда при реєстрації в gid додає імя користувача
  • git config --global user.email blabal@bla команда при реєстрації в gid додає електронну пошту користувача
  • git init- створює локальний репозиторій
  • git status - просмотр текущих изменений
  • git add - добавить файл в локальный репозиторий
  • git commit - коммит в локальный репозиторий
  • git log- показує образно історію репозиторія що з ним відбувалось
  • git checkout- відміняє зміну файла
  • git branch- команда дозволяє створювати, переглядати, перейменувати, удаляти гілки репозиторія
  • git push - коммит в удаленный репозиторий
  • git pull- команда зазвичай дістає дані з серверу, з якого ви зробили клон, та намагається злити її з кодом, над яким ви зараз працюєте
  • git merge- используется для слияния одной или нескольких веток в текущую.
  • git clone - скачивание репозитория
  • .gitignore- Файл із розширенням файлу GITIGNORE - це файл Git Ignore, який використовується у системі управління версією / джерелом під назвою Git. Він визначає, git файли та папки не слід ігнорувати у заданому вихідному коді.
